# 🌌 SUC SOFTWARES - EXTERNAL BYPASS

O **SUC SOFTWARES** é um sistema de injeção externa via hardware (USB) focado em sigilo total. 
Zero modificação de APK, zero arquivos no celular.

### 🚀 Como usar:
1. Abra o **Painel Web** no celular (GitHub Pages).
2. Execute o `suc_injector.py` no PC.
3. Conecte o cabo USB ao celular (com Root/ADB ativo).
4. Ative as funções no site.
5. **BYPASS:** Puxe o cabo para limpar todos os logs instantaneamente.

### 🛡️ Segurança:
- **Auto Scan:** Encontra endereços de memória automaticamente.
- **Volatile Injection:** Nada é salvo no armazenamento físico.
- **Log Flush:** Limpa histórico de depuração ao desconectar.
